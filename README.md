# 🌿 Plant Disease Detection using Deep Learning

An AI-powered plant disease detection system that classifies plant leaves as **healthy or unhealthy** using a deep learning model. This project aims to assist farmers and agricultural experts in early disease detection and crop protection.

---

## 🚀 Features

* 🌱 Detects plant health from leaf images
* 🧠 Deep learning model (DenseNet-121)
* ⚡ Fast and accurate predictions
* 📊 Binary classification: Healthy vs Unhealthy
* 📱 Can be integrated into mobile apps

---

## 🧠 Model Details

* Model: DenseNet-121
* Framework: TensorFlow / PyTorch
* Input: Leaf images
* Output: Healthy / Unhealthy classification

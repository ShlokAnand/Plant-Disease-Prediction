# 🌿 Plant Disease Detection System

A deep learning-based application that detects plant diseases from leaf images. The system classifies plants as **healthy or unhealthy**, helping in early diagnosis and improving agricultural productivity.

---

## 🚀 Features

* 🌱 Leaf-based disease detection
* 🧠 Deep learning model (DenseNet-121)
* 📊 Accurate classification (Healthy / Unhealthy)
* ⚡ Fast prediction results
* 📁 Supports image input from dataset or user

---

## 🧠 Model Details

* Model: DenseNet-121
* Framework: TensorFlow / PyTorch
* Input: Leaf images
* Output: Healthy / Unhealthy
